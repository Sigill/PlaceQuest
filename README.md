# PlaceQuest

PlaceQuest is a database + webpage to help you in your quest for a new place to live.

## Install

```sh
bundle install --path ./vendor
bundle exec ./vendor/bundle/ruby/2.5.0/bin/rerun 'rackup config.ru -o 0.0.0.0 -p 4567'

```
